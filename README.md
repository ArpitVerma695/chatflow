# 💬 ChatFlow

A modern real-time chat application built using React, Node.js, Express, MongoDB, and Socket.io.

## 🚀 Features

- Real-time messaging
- Username login
- Chat history
- Typing indicator
- Online users
- Emoji support
- Dark mode
- Responsive UI

## 🛠️ Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- Socket.io
- Tailwind CSS

## ▶️ Run Project

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
